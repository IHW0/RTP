# RTP

Плагин для Minecraft сервера, добавляющий команду случайной телепортации игроков в безопасное место.

## 🎯 Возможности

- Случайная телепортация игрока по команде
- Поиск безопасного места для телепортации
- Настраиваемый радиус RTP
- Сообщения с цветовым оформлением
- Поддержка разрешений для команды
- Команда доступна обычным игрокам при наличии permission

## 📋 Требования

- Minecraft Server 1.21+
- Java 17+
- Maven 3.6+

## 💬 Команды

| Команда | Описание | Разрешение |
|---|---|---|
| `/rtp` | Случайная телепортация | `rtp.use` |
| `/rtp help` | Показать справку | `rtp.use` |

## 🔐 Разрешения

`rtp.use` — разрешение для использования команды `/rtp`

## 🔧 Использование

1. Введите команду `/rtp` в чат
2. Плагин начнёт поиск безопасного места
3. После нахождения подходящей точки игрок будет телепортирован
4. В чат придёт сообщение об успешной телепортации

## ⚙️ Настройка прав

Чтобы все игроки могли использовать команду, добавьте permission через LuckPerms:

```bash
/lp group default permission set rtp.use true
